## Questions (KPIs)
-	Attrition percent in each Age group.
-	Calculate gender based Attrition percent.
-	Attrition percent in each Marital Status.
-	Attrition percent with over time worker.
-	Attrition percent in each department.
-	Calculate Attrition percent in each Education field.
-	Calculate Attrition percent in each job role.
-	Environment Satisfaction in each department.
-	Number of Job Role in each department.
-	Department wise Salary Hike percent.
-	Department wise Training Time in last Year.
-	Make plot of Performance Rating and Job Involvement in each Job Role.
-	Make Work life balance plot with Department and Over Time .
-	On the bases of Last Year Training Time make plot with Performance Rating and Job Satisfaction.
-	Dashboard Interaction <a href="https://github.com/saurabhtiwari-ctr/HR-Analytics-Dashboard/blob/main/HR%20Dashboard.pdf"  >View Dashboard</a>

## Process
-	Verify data for duplicate value , improper data , common column and extra space.
-	Make sure data is consistent and cleaned with respect to data type, format and value used.
-	Add many column at requirement based.
-	Create plot in Tableau on the bases of question asked.
-	Merge all plot into one dashboard to give detail information at one place.
